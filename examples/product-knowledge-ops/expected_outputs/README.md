# Expected Outputs

This folder documents the expected shape after deterministic tools run on the example.

- `ai_kb/wiki/source_manifest.md` lists all three raw sources.
- `ai_kb/wiki/source_cards/` contains one card per raw source.
- `ai_kb/wiki/current/` contains confirmed Core Knowledge Portal state.
- `ai_kb/wiki/current_draft/` contains draft Flexible Review Workflow state.
- `ai_kb/export_for_ai/mini_kb/` contains a review-prep mini-kb.
